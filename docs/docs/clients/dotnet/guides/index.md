# .NET Platform Guides

This section provides guides for setting up and configuring Exceptionless with common .NET application types. See [.NET Packages and Platforms](/docs/clients/dotnet/supported-platforms) for the complete .NET package list, or browse the [client directory](/docs/clients/) for other languages and runtimes.

Exceptionless tries to provide support for platforms as long as Microsoft supports them. As such, you will be able to utilize any of our packages for your applications, but this documentation will focus on our core packages for .NET.
