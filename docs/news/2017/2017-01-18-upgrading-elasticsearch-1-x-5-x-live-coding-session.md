# Upgrading from Elasticsearch 1.x to 5.x - Live Coding Session

[![elasticsearch 1.x to 1.5 upgrade](/assets/img/news/live-stream-1-elasticsearch-header-1024x538.jpg)](https://www.liveedu.tv/niemyjski/videos/xAYyE-exceptionless-weekly-demo-1-9-17)

## New Weekly Coding Live Stream

In the first episode our **all new** [weekly live streaming demo session](https://www.liveedu.tv/niemyjski/videos/xAYyE-exceptionless-weekly-demo-1-9-17), Blake talks about the recent push to migrate Exceptionless from Elasticsearch 1.x to Elasticsearch 5.x.

One major part of this project has been updating [Foundatio.Repositories](https://github.com/FoundatioFx/Foundatio.Repositories) and [Foundatio.parsers](https://github.com/FoundatioFx/Foundatio.Parsers), both of which are parts of our [pluggable foundation block project for building distributed apps](https://github.com/FoundatioFx/Foundatio), to support Elasticsearch 5.x.

Once that was complete, we then worked on reindex scripts for migrating data from Elasticsearch 1.x to 5.x using [Elasticsearch's external reindex](https://www.elastic.co/guide/en/elasticsearch/reference/5.1/docs-reindex.html).

Check out the live stream, and tune in weekly for the next episode.
